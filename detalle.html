Programación - Desarrollo de página web

Fecha de inicio: 2026

Autores: Lourdes Aramayo y equipo

Descripción

Este proyecto tiene como objetivo enseñar a los estudiantes a desarrollar páginas web utilizando HTML, comprendiendo la estructura básica de un sitio y el uso de etiquetas semánticas.

Además, se busca fomentar habilidades digitales, el pensamiento lógico y la creatividad mediante la creación de contenido web dentro de un entorno educativo.
